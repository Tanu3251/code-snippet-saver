# Code Snippet Saver

## Project Description
Code Snippet Saver is a repository used to store reusable programming code snippets. Developers often reuse small pieces of code such as sorting algorithms or search functions in different projects.

## Objective
The objective of this project is to demonstrate how Git version control can be used to maintain and track the history of saved code snippets.

## Features
- Stores useful programming snippets
- Organizes snippets by programming language
- Tracks changes using Git commits
- Maintains version history

## Snippets Included
- Bubble Sort (Python)
- Binary Search (Python)
- Factorial (C)

## Version Control
Git is used to track modifications to the snippets. Each update creates a new commit which allows developers to view previous versions and track improvements over time.
